## Dependencies / Prerequisites

### packages :

- hyprland
- hyprland-lua
- hyprpaper
- hypridle
- hyprlock
- hyprshot
- hyprshade
  
- waybar
- rofi
- mako
  
- kitty
- zsh
- oh-my-zsh
- powerlevel10k
  
- yazi
- nautilus
- vscodium
  
- wl-clipboard (`wl-copy`, `wl-paste`)
- cliphist
  
- playerctl
- pavucontrol
- wireplumber (`wpctl`)
- brightnessctl
- libnotify (`notify-send`)
  
- pywal (`wal`)

- eza/exa
- fastfetch
- tty-clock
  
- Bibata cursor theme
- GeistMono Nerd Font Mono

on Arch, `yay`/`pacman` can be used to install most of them.
